# CodeAlpha_Music_System_Frontend
# 🎵 Music Player System

A modern, responsive, and sleek web-based Music Player application built using core web technologies (**HTML5**, **CSS3 Custom Variables**, and **Vanilla JavaScript**). The interface leverages a dark-themed glassmorphic aesthetic to deliver a smooth and visually pleasing audio streaming simulation directly in the browser.

---

## 🚀 Features

* **Custom Audio Controls:** Seamless play, pause, next, and previous track actions.
* **Dynamic Metadata Tracking:** Real-time updates for track title, artist name, cover art, elapsed time, and total duration.
* **Interactive Scrubber:** A responsive track progress bar allowing users to jump to any part of the song.
* **Smart Volume Slider:** Custom volume adjustments with dynamic indicator icon updates (Mute / Low / High).
* **Bonus Features Added:**
    * **Interactive Queue/Playlist:** Click directly on any song in the playlist to change tracks immediately.
    * **Gapless Autoplay:** Automatically shifts to the next track in the queue once the current track ends.

---

## 🛠️ Tech Stack

* **Structure:** HTML5 (Semantic elements)
* **Styling:** CSS3 (Flexbox, Glassmorphism gradients, Custom range inputs)
* **Functionality:** Vanilla JavaScript (HTML5 `Audio` API)
* **Icons:** FontAwesome v6.4.0 (via CDN)
